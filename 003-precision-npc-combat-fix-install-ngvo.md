# Instalacja Precision - NPC Combat Fix w NGVO

- Ta instrukcja zakłada, że masz już zrobione `001` i `002`.
- Pobierz mod `Precision - NPC Combat Fix` do MO2:
  - https://www.nexusmods.com/skyrimspecialedition/mods/169711
- Zainstaluj mod w MO2.
- Włącz mod w lewym panelu MO2.
- Uruchom `Pandora Behaviour Engine Plus`.
- Wygeneruj behaviory ponownie.
